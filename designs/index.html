⚡ Impeccable Evaluation

pabakaus/impeccable (42K ★) — a design guidance system for AI coding tools. We installed it in the repo at .impeccable/.

✅ Can be used here Impeccable provides design context files (DESIGN.md + PRODUCT.md), 23 commands, 44 anti-pattern detectors, and a design vocabulary usable by any AI agent working on this project.
🎨 Design 3 uses it Industrial Teal directly applies Impeccable's kinpaku tokens: lacquer-black, verdigris-patina accent, weight-300 uppercase display, zero radius, compressed hierarchy.
📋 How to use Add Impeccable's DESIGN.md + PRODUCT.md as design context. Run commands via AI tool integration (Claude Code, Cursor, Codex). Also works standalone via AGENTS.md reference.
📐 Anti-pattern rules 44 deterministic detector rules help catch: gray text on colored backgrounds, cards-inside-cards, overused fonts, bounce easing, pure black/gray without tinting, and more.

Verdict: Impeccable is installable and applicable. Its DESIGN.md tokens directly inspired Design 3 (Industrial Teal). The commands (critique, polish, audit, bolder, quieter) work through AI tool hooks. For standalone use, the anti-pattern detector CLI can be run directly.
